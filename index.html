Instantiate a clean spreadsheet worksheet at sheets.new. Title the log document: AHSU Live Engine.

Inject Cloud Macro Script

3
Deploy Automation Endpoint
Click Deploy → New deployment. Select Web app template configuration. Execute role as "Me", and grant authorization criteria privileges to "Anyone". Copy production microservice route URL.
