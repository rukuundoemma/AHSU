One-Time Setup (5 minutes)
1
Create a Google Sheet
Go to sheets.new to create a new Google Spreadsheet. Name it AHSU Attendance.
2
Open Apps Script
In the spreadsheet, click Extensions → Apps Script. Delete any existing code, then paste the code below and click Save (Ctrl+S).
3
Deploy as Web App
In Apps Script, click Deploy → New deployment. Set type to Web app, set Execute as to Me, set Who has access to Anyone. Click Deploy and approve permissions. Copy the Web App URL.
4
Paste Your Web App URL Here
Paste the deployment URL below. This connects the attendance system to your Google Sheet.

How to Share Access

1. Create a department above with a code and PIN.
2. Share this URL with the department admin — they enter their code + PIN to sign in:

https://rue-ahsuattendance.netlify.app/

3. Tell them their code and PIN separately (e.g. via phone).
4. They manage their own residents, sessions, history — completely isolated from yours.
5. You can reset their PIN or remove them any time from here.
